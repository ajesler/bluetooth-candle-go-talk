GATT

Generic Attribute Profile

Client

Server

Characteristic

Service

Descriptor

Client: A device that initiates GATT commands and requests, and accepts responses, for example, a computer or smartphone Server: A device that receives GATT commands and requests, and returns responses, for example, a temperature sensor. Characteristic: A data value transferred between client and server, for example, the current battery voltage. Can read and write. Service: A collection of related characteristics, which operate together to perform a particular function. eg heart rate service, battery, device name Descriptor: A descriptor provides additional information about a characteristic. Can be multiple. Optional. eg a unit of measure for a characteristic.
